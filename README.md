# Ecommerce Sales Analysis 

## Overview
Exploratory Data Analysis (EDA) of 5,000 ecommerce transactions spanning October 2023 to September 2025. This project uncovers sales trends, top-performing categories, regional profit patterns, and customer payment behavior.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- **Source:** [E-Commerce Sales - Kaggle](https://www.kaggle.com/datasets/prince7489/e-commerce-sales)
- **Records:** 5,000 transactions
- **Period:** October 2023 - September 2025 (October 2025 excluded from monthly trend - incomplete data, only 3 days)
- **Features:** 14 columns including Order ID, Category, Region, Sales, Profit, Payment Mode

## Key Findings
- **Home Decor** is the top-selling category, narrowly ahead of Furniture and Clothing
- **May 2025** was the best performing month
- **North Region** generates the highest profit
- **Net Banking** is the most-used payment method, though usage is fairly balanced across all five modes
- **Headphones Accusantium** is the most profitable product
